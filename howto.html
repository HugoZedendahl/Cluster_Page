How to use Cluster Pages

Cluster Pages is a way to store and accsess different web sessions quickly and easily. Being a new take on how a home page sould work.

set up

Turn off adblock and accept all popups from the cluster pages site. If you dont do this the clusters will very likely be blocked from opening

What are clusters?

A cluster on Cluster Page is a collection of links that you can launch simultaniously by clicking on a cluster. the idea is that you store multiple pages in different clusters depending on what you are going to do at the start of each web browsing session

How Do You Store links?

go to the settings page and write or paste the URL link to the website you want to add to that cluster and press the submit button below the input field. the clusters are ordered numerically from low to high, left to right. cluster one is therefore the top left cluster, and cluster six is the bottom right cluster.

Naming a Cluster?

similar to how you add links to a cluster you can name your clusters. go to the settings page and enter the name you want to be displayed at the top of the cluster in the change name field and press submit.

resetting a cluster?

to reset a cluster you will have to go to the settings page check the checkbox in the cluster. after you checked that checkpoint and click submit you will clear all data from that cluster, removing all links and removing the name

where does my data go?

all user generated data is stored locally in your browser meaning that all data to all clusters never leave your computer. if you clear the cash and other data off your browser you will clear all the clusters.
